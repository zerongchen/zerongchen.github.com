Memory-EfficientJavaCode

2021-04-14 评论

初识 java对象内存

int t=4; 在java中需要多少个位表示int？ 4个byte ，32bit
但是Integer对象呢，需要多少位呢？

答案是128位，也就是int的4倍（32位系统）
4 8 12 16
|- - - - |- - - - |- - - - |- - - - |
|class point | flags | lock info | value = 4 |

64bit 系统需要 28字节 共 224
8 16 24 28
|- - - - |- - - - |- - - - |- - - - |
|class point | flags | lock info | value = 3 |
